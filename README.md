# Pytest Trading Simulator Tests

This repository contains Python + pytest automated tests for a trading simulator.
It demonstrates API testing, database access, SSH log parsing, and integration scenarios.

## Features
- REST API tests
- DB queries (PostgreSQL)
- SSH log reading
- Algorithm trace validation
- Assertions and utilities
- Pytest fixtures

## How to run
1. Set environment variables (optional, defaults provided)
2. Install dependencies: pip install -r requirements.txt
3. Run tests: pytest -v
